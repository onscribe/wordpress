== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.


== Frequently Asked Questions ==

= Can I customize my buttons? =

Yes, in the settings page

= Will this plugin work with WordPress versions less than 3.9? =

There is a good chance it will work on versions 3.x. Although we haven't tested it, the plugin is using standard methods well established before v3.9. Use at your own risk.
